# 💰 Expense Tracker

Expense Tracker is a full-stack web application designed to help users track, manage, and organize their daily expenses efficiently.

The application provides a simple and intuitive platform where users can record expenses, categorize spending, maintain financial records, and gain better insights into their spending habits.

---

# 📖 Overview

Managing personal finances is an important part of everyday life. Expense Tracker helps users maintain a digital record of their expenses, making it easier to monitor spending and manage budgets effectively.

The application provides an organized interface for recording transactions and viewing expense data in a structured manner.

# ⚙️ Installation & Setup

## 1. Clone the Repository

```bash
git clone https://github.com/VanshMirani/Expense-Tracker.git
```

## 2. Navigate to Project Directory

```bash
cd Expense-Tracker
```

## 3. Install Dependencies

```bash
npm install
```

## 4. Configure Environment Variables

Create a `.env` file and add:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```

## 5. Start the Backend Server

```bash
npm start
```

or

```bash
npm run server
```

## 6. Start the Frontend

```bash
npm start
```

## 7. Open the Application

```text
http://localhost:3000
```
